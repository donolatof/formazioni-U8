# Formazioni U8

Benvenuti nell'app Formazioni U8!

## Descrizione
L'app Formazioni U8 consente di gestire le formazioni calcistiche per le squadre giovanili di calcio. È progettata per offrire un'interfaccia semplice e intuitiva per creare e visualizzare le formazioni, tenere traccia dei giocatori e delle loro posizioni.

## Caratteristiche
- Creazione e modifica delle formazioni
- Visualizzazione delle posizioni dei giocatori
- Supporto per diverse formazioni (4-4-2, 3-5-2, ecc.)
- Possibilità di salvare e caricare formazioni
- Interfaccia utente intuitiva e facile da usare

## Installazione
1. Clona il repository:
   ```bash
   git clone https://github.com/donolatof/formazioni-U8.git
   ```
2. Naviga nella cartella del progetto:
   ```bash
   cd formazioni-U8
   ```
3. Installa le dipendenze:
   ```bash
   npm install
   ```

## Utilizzo
Dopo aver installato le dipendenze, puoi avviare l'applicazione:
```bash
npm start
```
